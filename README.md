# Personal Portfolio

A modern, responsive personal portfolio built using **Next.js** to showcase my skills, projects, and professional presence as a software engineer.

This portfolio is designed with a clean dark UI, smooth animations, and performance-focused architecture, making it suitable for both recruiters and developers.

---

## Overview

This project serves as my personal portfolio website, highlighting:
- Professional experience
- Technical skills
- Projects
- Social and contact information

The application is optimized for performance, accessibility, and responsiveness across all devices.

---

## Tech Stack

- **Framework:** Next.js  
- **Language:** TypeScript / JavaScript  
- **Frontend:** React  
- **Styling:** Tailwind CSS  
- **Animations:** Framer Motion  
- **Icons:** React Icons, Lucide React  
- **Package Manager:** pnpm / npm  
- **Deployment:** Vercel / Netlify  

---

## Features

- Fully responsive design
- Modern dark-themed UI
- Smooth scroll and animation effects
- Modular and reusable components
- SEO-friendly Next.js structure
- Optimized asset loading
- Easy content management via data files


---

## Getting Started

### Prerequisites

- pnpm or npm
